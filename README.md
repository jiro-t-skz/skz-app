# Overview
- 櫻坂46の非公式コミュニティサイトのポートフォリオです。
- 『推しメンの写真が当たらない』、『一緒にライブで楽しんでくれる人を探したい』などの悩みを解決する機能を実装しました。
- 投稿、オフ会、同行者募集・トレードの3つの投稿機能を切り替えて使用します。
- プログラミングゼミのポートフォリオの完成版です。
