# Overview
- 櫻坂46の非公式コミュニティサイトのMVPです。
- 『推しメンの写真が当たらない』、『一緒にライブで楽しんでくれる人を探したい』などの悩みを解決する機能を実装しました。
- タブの切り変えて3つある投稿機能を切り替えます。※MVPなので、トレード・同行者募集機能のみ。色も白黒ベース。
- プログラミングゼミのMVPの課題として作成しています。
